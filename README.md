## Aplicacion calculadora

ejecutar en la raiz del repo
```
mvn clean package
java -jar .\target\rest-service-0.0.1-SNAPSHOT.jar
```


### Ejemplo 1
**input**
{
	"entrada" : "5+9"
}

**output**
{
	"input" : "5+9",
	"output" : "14.0"	
}
